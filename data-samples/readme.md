# Data Samples

Here, I present two files with the identical Scancode Maps I actually use. It remaps certain keys:

- The keypad characters “0” to “9” and the keypad decimal point are remapped to the navigation keys, Insert and Delete.
In other words, the numeric keypad is disabled and replaced with the navigation and clipboard functions,
the same functions applied when Num Lock is off.
- Num Lock is remapped to Context Menu.
- Caps Lock is remapped to Context Menu.

It solves the problem if using the keyboard blindly. The Num Lock state is confusing and leads to many mistakes.
Using the keypad navigation is very convenient, but accidentally pressing Num Lock breaks the workflow.

Another solved problem is that on many modern keyboards, one of the most important keys, Context Menu, is eliminated.
My remapping makes it available for all keyboards. The loss of Caps Lock only improves typing.
